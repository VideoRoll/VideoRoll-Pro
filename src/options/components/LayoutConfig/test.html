💾 Save-on-Demand Features

  • Change Tracking: System tracks all configuration changes and maintains a snapshot of saved state
  • Visual Feedback: Save button changes color (orange) and shows badge when unsaved changes exist
  • Status Indicators: Header shows "未保存" (Unsaved) badge and reminder text
  • Keyboard Shortcuts: Ctrl+S saves configuration when changes are pending
  • Leave Protection: Browser shows confirmation dialog when leaving with unsaved changes

🎨 User Interface Features

  • Tab Configuration: Toggle visibility of video, audio, download, and settings tabs
  • Component Management: Show/hide individual components with eye icons
  • Layout Customization: Adjust rows, columns, and component widths
  • Drag & Drop: Reorder components by dragging between rows and tabs
  • Preset Layouts: Quick apply compact, spacious, or single-column layouts
  • Export/Import: Save and restore configuration as JSON files

🚀 How to Use

  1. Access Configuration: Go to Extension Options → Layout Configuration
  2. Customize Layout: Toggle components, adjust widths, drag to reorder
  3. Save Changes: Click the "Save" button to persist your changes
  4. Apply Presets: Use quick preset buttons for common layouts
  5. Export/Import: Share configurations or backup your settings
